# Sidebar Menu UI 

A clean and modern sidebar navigation UI built using HTML and CSS. This project demonstrates how to create an interactive, toggleable sidebar menu using only CSS (no JavaScript), with smooth transitions and structured layout.

---

## Features  

### Sidebar Navigation  
#### - Toggleable sidebar using pure CSS (checkbox technique)  
   - Smooth slide-in and slide-out animation  
   - Fixed position sidebar  

### UI Components  
- Navigation menu with icons (Gallery, Events, Store, etc.)  
- Social media icons section  
- Logo/header section inside sidebar  

### Styling & Effects  
- Full-screen background image  
- Hover effects on menu items and icons  
- Smooth transitions for better UI experience  

### Responsiveness  
- Works across different screen sizes  
- Flexible layout using CSS  

---

## Technology Stack  

- HTML5  
- CSS3 (Transitions, Positioning)  
- Font Awesome (icons)  
- Google Fonts (Poppins)
