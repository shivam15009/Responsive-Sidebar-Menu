# Sidebar Menu UI 

### A clean and modern sidebar navigation UI built using HTML and CSS. This project demonstrates how to create an interactive sidebar menu using only CSS (no JavaScript), with smooth transitions and a structured layout.

## Features

#### 1. Sidebar Navigation
Toggleable sidebar using pure CSS (checkbox hack)
Smooth slide-in and slide-out animation
Fixed position sidebar

#### 3. UI Components
Navigation menu with icons (Gallery, Events, Store, etc.)
Social media icons section
Logo/header inside sidebar

#### 5. Styling and Effects
Full-screen background image
Hover effects on menu items and icons
Smooth transitions for better user experience

#### 7. Responsive Design
Works across different screen sizes
Flexible layout using modern CSS

##  Technology Stack
#### HTML5
#### CSS3

## How It Works
 Uses a hidden checkbox (#check) to control sidebar visibility
  When checked, CSS moves the sidebar into view
  No JavaScript is used
